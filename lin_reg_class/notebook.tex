We will use Linear regression to demonstrate Moore's law: Fig moore's
law

Here the problem is exponential as every year the number of transisters
double. so we will use log of number of transistors to make it linear.
number of transisters is in 2nd column and year is in 3rd. Goal here
is:(i)to show linear relationship in log of number of transisters and
year.(ii)to show number of transistors double every two years

taking partial derivative of squared error wrt a and b respectively, we
solve for a and b.
